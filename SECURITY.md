## Reporting a Vulnerability



We take the security of this project seriously. If you discover a security vulnerability, please report it privately. **Do not report security vulnerabilities through public GitHub issues.**

Please send an email to **[sajanpandit585@gmail.com]** with details about the vulnerability. 

**What to expect:**
* We will acknowledge receipt of your vulnerability report within 48 hours.
* We will send you regular updates on our progress in patching the issue.
* Once the issue is resolved, we will notify you before public disclosure.
